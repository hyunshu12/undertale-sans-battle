# Slice 4-5 정밀 구현 스펙 (BTS 소스 리버스엔지니어링)

> VM(vm.c)·CSV파서는 완성. 미구현 game command + BLUE물리 + 전투흐름 + 대사만 채우면 됨.
> 좌표 BTS 640×480 1:1, 각도=도(0우/90하/180좌/270상), dt=1/60, HP=MaxHP=92.

## 핵심 상수
- HeartSpeed=150 px/s (focus/Shift=75). RED 즉시세팅·BLUE 횡이동 공용.
- HEART_JUMP_STRENGTH=180 (anti-grav 엣지+grounded 임펄스). JUMPHOLD_CUTOFF=30 (가변점프컷: 비엣지 유지중 상승>30→-30).
- MaxFallSpeed 기본 750 (HeartMaxFallSpeed로 변경, SansSlam 속도).
- BLUE 중력 밴드(downSpeed=vx*cos(ang)+vy*sin(ang)): >240→540, 15~240→180, -30~15→450, -120~-30→180, ≤-120→180, else 0. grounded(아래1px solid) 아니면 적용.
- iframe=0.033s 쿨다운(깜빡임 아님, 재피격 락아웃).
- SansSlam: angle=floor(dir)*90, v=MaxFallSpeed; 착지 |v|≥330→shake(|v|/90)+SlamDamage시 hp-1.
- Platform: px/s 이동, reverse 경계반전 dir=(dir+2)%4, 위에서만 밟기, 시각선 y-4, 착지스냅 8.05.
- BoneStab: speed=Distance*10, Karma6, WARN→IN→HOLD(StayTime)→OUT, 진입깊이=Distance.
- SineBones(Count,Spacing,Speed,Height): Sine=floor(sin(i/3 rad)*28); Spacing>0→X=BBoxRight+Spacing*i,dir2; <0→X=BBoxLeft+Spacing*i,dir0; 상단본 BoneV(X,BBoxTop+6,Height+Sine,dir,Speed), 하단본 Y=상단끝+39 BoneV(X,Y,BBoxBottom-5-Y,...). Dmg1/Karma6.
- GasterBlaster(Size,SX,SY,EX,EY,EndAng,WaitTimer,BlastTime): ENTER p+=(target-p)*dt*10·|diff|<3스냅→WAIT(waitTimer)→FIRE(0.1s)→빔: baseSize=35*GBh/imgh(램프업 4/30s), blastTime후 0.8^(dt*30)감쇠, opacity=100-((beamTimer-blastTime)*30-5)*10, baseSize<2 destroy. 폭1000, 충돌높이=baseSize*3/4, opacity≥80만 Dmg1/Karma10. 충돌 OBB: 소울중심을 블래스터기준 -ang 역회전→로컬(lx,ly), 0≤lx≤1000 && |ly|≤h/2. LEAVE leaveSpeed+=30, x-=cos(ang)*dt*ls. Size 0~2 스케일.
- CombatZone ResizeSpeed 480 (CombatZoneSpeed로 변경), 4변 min(speed*dt,|cur-tgt|), 도착시 EndResize(보통 TLResume) 콜백.
- 전투흐름 임계: HitAttempts<13=INTRO, ==13=SPARE, 13<x<22=MULTI, ==22=FINAL.
  introTable[15], multiTable[10], 끝이면 choose 랜덤. Sans회피 X=320-cos(DodgeTimer*225)*100, DodgeTimer>0.4 종료시 HA++.
- KARMA: 피격 kr+=karma(상한40, kr≤hp-1), 드레인 interval=kr≥40?0.033:≥30?0.066:≥20?0.166:≥10?0.5:1.0초마다 kr--&hp--. AttackSprite Karma>3→2캡(뼈/빔 캡없음).
